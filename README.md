# 开发商风险项评估系统

基于公开数据源的房地产开发商风险项评估工具，支持独立项和参考项的自动化评估。

## 功能特性

- **公司搜索**：支持股票代码、简称、全称搜索
- **独立项评估**（5项）：
  - 公开市场债务实质性违约（严重）
  - 被监管/司法立案调查（严重）
  - 核心资产被司法查封/冻结（严重）
  - 实际控制人/控股股东变更（警示）
  - 项目实质性烂尾/长期停工（警示）
- **参考项评估**（3项）：
  - 高管频繁变动
  - 负面舆情集中爆发
  - 因违规被暂停网签或预售审批
- **报告生成**：在线查看 + PDF导出
- **历史记录**：自动保存评估历史

## 技术栈

- Python 3.10+
- Streamlit
- SQLite
- ReportLab（PDF生成）

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 部署

本项目支持部署到 [Streamlit Cloud](https://streamlit.io/cloud)。
